# A股港股行情查询

当用户询问股票行情、价格、涨跌时使用。

## 使用示例

- "茅台今天怎么样？"
- "600519 股价"
- "腾讯控股涨了还是跌了？"
- "9988.HK 行情"
- "帮我查一下平安银行"

## 调用方式

```bash
python3 scripts/cnstockinfo.py <股票代码>
```

## 参数格式

| 市场 | 代码格式 | 示例 |
|------|---------|------|
| A 股 | 6位纯数字 | `600519`（茅台）、`000001`（平安银行） |
| 港股 | 数字.HK | `9988.HK`（腾讯）、`3690.HK`（美团） |

## 输出示例

```
==================================================
  贵州茅台 (600519) [A股]
==================================================
  当前价: 1680.00 📈
  涨跌: +15.50 (+0.93%)
  开盘: 1675.00
  最高: 1690.00
  最低: 1670.00
  成交量: 3,500 万股
==================================================
```

## 数据来源

- A 股：新浪财经 `hq.sinajs.cn`
- 港股：东方财富 `push2.eastmoney.com`

## 注意事项

- 完全免费，无需 API key
- A 股代码可省略 sh/sz 前缀
- 港股代码不区分大小写（.HK 或 .hk 都可以）
